# Proposed Plan

**1. Filter Panel (Search Panel) Enhancements**
- **Panel Collapse:** I will introduce a master collapse/expand button in the header of the `SearchPanel`. When collapsed, it will shrink to a tiny icon button on the left edge, freeing up your view of the canvas.
- **20% Default Width:** I will set the default CSS width of the expanded panel to `w-1/5` (20% of screen width), while retaining the `min-w-[20rem]` safeguard and your ability to manually resize it.
- **Parent Dimension Default:** I will add a `defaultCollapsed` prop to the internal `DimensionFilter` component and set it to `true` specifically for the Parent dimension so it stays out of the way until needed.

**2. Asset Details (Sidebar) Enhancements**
- **Fix Top Bar Overlap:** I will change the CSS positioning of the `Sidebar` from `fixed` to `absolute` relative to the `<main>` content container. This guarantees it fits perfectly underneath the top header bar rather than covering it up.
- **JSON View Controls:** 
  - I will change the default rendering of the `@uiw/react-json-view` component to be fully expanded (`collapsed={false}`).
  - I will add a small "Collapse All" / "Expand All" toggle button to the "Raw Data" section header so you can quickly fold or unfold massive JSON payloads with a single click.
