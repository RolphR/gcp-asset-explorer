# Plan: Add Zoom To Button to Asset Details Panel

1. **State Management (`App.tsx`)**:
   - Introduce a new numerical state variable `const [zoomTrigger, setZoomTrigger] = useState(0);`.
   - Create a `handleZoomTo` function that increments this trigger state:
     ```tsx
     const handleZoomTo = () => setZoomTrigger(prev => prev + 1);
     ```
   - Pass `zoomTrigger={zoomTrigger}` to the `<GraphViewer>` component.
   - Pass `onZoomTo={handleZoomTo}` to the `<Sidebar>` component.

2. **Graph Behavior (`GraphViewer.tsx`)**:
   - Update `GraphViewerProps` to include `zoomTrigger?: number`.
   - Add a new `useEffect` hook that watches `zoomTrigger`.
   - When the trigger changes and a `selectedNodeId` is present, look up the node's coordinates.
   - Call both `fgRef.current.centerAt(node.x, node.y, 800)` and `fgRef.current.zoom(4, 800)` to force the camera to pan to the node and reset its zoom level to 4.

3. **User Interface (`Sidebar.tsx`)**:
   - Import the `Focus` icon from `lucide-react`.
   - Add `onZoomTo: () => void;` to `SidebarProps`.
   - In the top-right button group of the Sidebar header, add a new interactive button next to the Next/Previous chevrons.
   - When clicked, this button triggers `onZoomTo()`.
