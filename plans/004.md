# Proposed Plan

**1. Data Parsing Updates (`src/utils/parser.ts`)**
- I will modify the `GraphNode` interface to explicitly hold three new pre-calculated properties: `cleanDisplayName`, `location`, and `assetType`.
- During parsing:
  - I will extract the location from `asset.resource.location` (defaulting to `"global"` if not present).
  - I will strip out the noisy prefix from the raw display name by finding the last `/` character and keeping only the text after it.

**2. Canvas Drawing Updates (`src/components/GraphViewer.tsx`)**
- I will rewrite the `nodeCanvasObject` custom drawing logic to center the text and its translucent background box exactly at the node's coordinate `(node.x, node.y)`, rather than offset below it.
- The text will be drawn as a three-line stack:
  1. **Display Name:** Bold, standard font size.
  2. **Location:** Slightly smaller font size, slate-gray color.
  3. **Asset Type:** Same smaller font size, slate-gray color.
- The color-coded node circle will still be drawn beneath the text box to anchor the links, but the text box will overlay it neatly.
