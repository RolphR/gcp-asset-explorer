   - I will modify `Sidebar.tsx` to conditionally render the custom properties `routerType`, `cidr`, `destRange`, and `status` inside the `Asset Metadata` table whenever they are non-empty for the selected `node`.

2. **References as Clickable Links**: 
   - I will add a new row to the `Asset Metadata` table in `Sidebar.tsx` for `relatedAssets`. 
   - These references will be displayed as an unordered list of clickable buttons/links.
   - I will add `onSelectNode: (id: string) => void` to `SidebarProps`.
   - In `App.tsx`, I will implement `handleSelectNode` which accepts an `id`, looks up the node in `data.nodes`, and calls `setSelectedNode(nextNode)`. 
   - I will pass this `handleSelectNode` down as `onSelectNode` to the `Sidebar`.

3. **Format Copied Strings (Remove Outer Quotes)**:
   - I will update the `<JsonView />` component call in `Sidebar.tsx` to include the `beforeCopy` prop.
   - The handler will be `beforeCopy={(text, keyName, value) => typeof value === 'string' ? value : text}`. This will ensure that string values inside the JSON viewer do not have outer double quotes when the copy button is clicked.

Add:
```tsx
  const handleSelectNode = (id: string) => {
    if (!data) return;
    const nextNode = data.nodes.find(n => n.id === id);
    if (nextNode) {
      setSelectedNode(nextNode);
    }
  };
```
