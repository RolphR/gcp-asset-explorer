**Here's what changed:**
1. I implemented the `nodeCanvasObject` custom drawing function for `react-force-graph-2d`.
2. Nodes are still rendered as circles, color-coded based on their service.
3. Beneath each node, it now draws a clean text block that includes:
   - **Display Name** in bold text (dark gray).
   - **Asset Type** below it in slightly smaller text (slate gray).
4. **Performance optimization**: To prevent the graph from becoming an unreadable, overlapping mess of text when there are thousands of nodes, the text labels will dynamically hide if you zoom out too far (`globalScale < 1.5`). As you zoom in to inspect areas of the graph, the display names and types will seamlessly fade in.
